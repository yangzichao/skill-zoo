# skill-zoo

A menagerie of weird and miscellaneous Claude Code skills.

## House rule

New skills must conform to **skill-spec** — the canonical anatomy (folder layout, `SKILL.md` shape, frontmatter, progressive disclosure). When authoring a new skill, activate `skill-spec` first.

## Skills

- [`skill-spec`](skills/skill-spec/) — the spec itself; what every other skill in this repo should follow.
- [`justin-sun-perspective`](skills/justin-sun-perspective/) — a multi-variable competition lens for finding leverage inside a system (industry, market, org, platform, community, network) and producing 1–3 high-leverage next moves. Also reframes single-variable attributions and defensive explanations into action maps. Morally neutral; refuses paths requiring fraud, illegality, or harm to others.
